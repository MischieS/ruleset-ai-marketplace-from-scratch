# Ruleset AI UI Studio (Next.js)

Frontend-first product workspace focused on:

- premium visual design
- business-driven information architecture
- role-based UX surfaces with mock data

## Routes

- `/` Home
- `/marketplace`
- `/user`
- `/seller`
- `/admin`
- `/business`

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Quality checks

```bash
npm run lint
npm run build
```

## Notes

This app is intentionally mock-data based so UI/UX and business workflow quality can be finalized before backend integration.
